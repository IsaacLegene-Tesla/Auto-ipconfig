# Baselines

One SCALANCE export per AMR.

**Easiest:** `Launch_Fleet_Config.bat` → **Baselines** → enter AMR # and upload the `.conf`.

Files are saved as `config_SCALANCE_S600_AMR_<#>.conf` and `inventory.csv` is updated automatically.

**Manual:** copy files here, then `python generator/extract_inventory.py`

These files are large and are gitignored (`.conf` in this folder).
